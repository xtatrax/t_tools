# VScode で開くの作り方

Macのアプリ一覧から Automator を開く 

    -> クイックアクションを選択 

    -> ワークフローが受け取る現在の項目: ファイル又はフォルダ を選択 

    -> 検索対象: Finder.app を選択 

    -> ←のバーから

        -> ライブラリ

            -> ユーティリティ

                -> シェルスクリプトを実行 を →領域へ D&D

                    -> 入力の引き渡し方法: 引数として を選択

                    -> 入力ボックスに下を入力
``` bash
open -a "Visual Studio Code" "$@" 
```
